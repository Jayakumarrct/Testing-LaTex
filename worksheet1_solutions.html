SA-1 Revision Worksheet 1: Step-by-Step Solutions

1. Solve 2x^2 - 10x + 5 = 0 (correct to 3 significant figures)

Discriminant Δ = b^2 - 4ac = (-10)^2 - 4(2)(5) = 100 - 40 = 60.
Roots x = (-b ± √Δ)/(2a) = (10 ± √60)/4 ≈ (10 ± 7.746)/4.
Thus,
  x1 = (10 + 7.746)/4 ≈ 4.44,
  x2 = (10 - 7.746)/4 ≈ 0.564.

2. Determine whether 5x^2 - 9x + 4 = 0 has real roots

Δ = (-9)^2 - 4(5)(4) = 81 - 80 = 1 > 0, so real roots exist.
Roots x = (9 ± √1)/10 = (9 ± 1)/10 = 1 or 0.8.

3. Solve 2x^2 + 4x - 8 = 0 (correct to one decimal place)

Δ = 4^2 - 4(2)(-8) = 16 + 64 = 80.
Roots x = (-4 ± √80)/4 = (-4 ± 8.944)/4.
  x1 = (-4 + 8.944)/4 ≈ 1.2,
  x2 = (-4 - 8.944)/4 ≈ -3.2.

4. Solve the inequality -9 ≤ 3 - 7x < 2 + x

From -9 ≤ 3 - 7x:
  subtract 3 → -12 ≤ -7x
  divide by -7 (flip sign) → x ≤ 12/7.
From 3 - 7x < 2 + x:
  subtract 3 → -7x < -1 + x
  subtract x → -8x < -1
  divide by -8 (flip sign) → x > 1/8.
Combined solution: 1/8 < x ≤ 12/7.

5. Solve the inequality x^2 ≤ 7 + x

Rearrange: x^2 - x - 7 ≤ 0.
Solve quadratic x^2 - x -7 = 0:
  x = (1 ± √(1 + 28))/2 = (1 ± √29)/2.
Roots ≈ -2.19 and 3.19.
Because the parabola opens upward, the inequality holds between the roots:
  (1 - √29)/2 ≤ x ≤ (1 + √29)/2.
Integer solutions: {-2, -1, 0, 1, 2, 3}.

6. Solve the inequality 3x ≥ 4 + x

3x ≥ 4 + x ⇒ 2x ≥ 4 ⇒ x ≥ 2.
Solution set: {x | x ≥ 2}.

7. Solve the inequality x^2 + 3x - 10 ≤ 0

Factor: x^2 + 3x - 10 = (x + 5)(x - 2).
For the product to be ≤ 0: -5 ≤ x ≤ 2.
Solution set: {x | -5 ≤ x ≤ 2}.
